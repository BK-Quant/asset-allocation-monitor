## 자산배분모니터 - 즉시 데이터 갱신

정기 자동 갱신(GitHub Actions, 매월말 19:00 KST 한국장 + 매월 1일 07:00 KST 미국장)과 별개로,
지금 이 순간의 최신 종가로 즉시 다시 계산하고 싶을 때 사용하는 수동 실행 스킬이다.

### 실행 순서

1. 프로젝트 루트(`퀀트/자산배분모니터/`)에서 아래 두 스크립트를 순서대로 실행한다.
   ```bash
   python scripts/update_data.py
   python scripts/build_output.py
   ```
2. `update_data.py`는 Yahoo Finance·BLS·FRED·us500.com에서 최신 가격·매크로 데이터를 받아 `data/prices.json`, `data/economic.json`을 갱신한다.
3. `build_output.py`는 "전월 말 종가 기준 확정 신호"(`data/current.json`)와 "월별 백테스트"(`data/backtests.json`)를 재계산한다.
   - 주의: 이 스크립트는 **오늘 날짜 기준으로 "직전에 완결된 달"**을 자동으로 판단한다. 즉, 월중에 이 스킬을 실행해도 "이번달 확정 배분"이 바뀌지는 않고(전월 말 종가 고정), 다만 "차월 예상(ADM 프리뷰)"과 "DAA 카나리아 프리뷰"만 최신 종가로 갱신된다. 이는 프로젝트의 핵심 설계 원칙(월중에 신호가 흔들리지 않음)이므로 정상 동작이다.
4. 두 스크립트 실행 후 출력 로그(기준일 → 적용월, 전략 수)를 확인해 정상 완료됐는지 보고한다.
5. `data/` 아래 변경사항이 있으면 `git status`로 보여주고, 커밋·푸시 여부를 사용자에게 물어본다(공개 저장소로 push하는 행위이므로 반드시 명시적 확인 후 진행 — 자동으로 push하지 않는다).

### 로컬 미리보기(선택)
확인을 원하면 `python -m http.server 4446`으로 로컬 서버를 띄우고 `http://localhost:4446`에서 결과를 직접 확인할 수 있다.
